# dd-6
データドリブン基礎の課題のためのログ読み取り
## 使用例
プロジェクトに何個かの使用例が付属している
### run.py
'0101'と'0102'の歩数記録をエクセルの表に出力する

output
|時間|0101歩数|0102歩数|
|----|-------|--------|
|12:00|0     |0       |
|12:01|0     |0       |
|12:02|0     |0       |
|12:03|0     |0       |
|... 
|0:45 |47    |0       |
|... 
### people.py
./dataに入っている全ユーザーの関係をポイントで表す。結果を一次元の表に入れる

output
|ユーザー１|ユーザー２|ポイント|
|----|----|----|
|0102|0101|227 |
|0103|0101|0   |
|0103|0102|0   |
|0104|0101|0   |
...
